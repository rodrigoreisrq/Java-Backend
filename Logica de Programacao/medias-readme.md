# Cálculo de Médias em Java

Este projeto foi desenvolvido em **Java** com o objetivo de calcular médias aritméticas e realizar operações simples utilizando variáveis inteiras.

## 📌 Objetivo

O programa realiza os seguintes cálculos:

1. Calcula a média dos números **8, 9 e 7**
2. Calcula a média dos números **4, 5 e 6**
3. Calcula a soma das duas médias
4. Calcula a média entre as médias obtidas

---

## 💻 Código utilizado

O programa utiliza:

- Variáveis do tipo `int`
- Operações matemáticas básicas:
  - Soma (`+`)
  - Divisão (`/`)
- Saída de dados com `System.out.println()`

---

## 🧮 Cálculos realizados

### Primeira média
Números:
8, 9 e 7

Cálculo:

(8 + 9 + 7) / 3

Resultado:

8

---

### Segunda média
Números:
4, 5 e 6

Cálculo:

(4 + 5 + 6) / 3

Resultado:

5

---

### Soma das médias

8 + 5

Resultado:

13

---

### Média das médias

(13 / 2)

Resultado:

6

---

## ▶ Como executar

1. Instale o Java JDK
2. Salve o código em um arquivo chamado:

```bash
Main.java
````

3. Compile o arquivo:

```bash
javac Main.java
```

4. Execute:

```bash
java Main
```
